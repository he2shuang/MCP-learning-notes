**使用 nvm 安装并切换到最新的 LTS 版本**

1. **重要**：**以管理员身份打开**一个新的 PowerShell 或 CMD 窗口。
2. 执行以下命令来安装最新的 LTS 版本：
    
    
    ```
    nvm install lts
    ```
    `nvm` 会自动找到并下载安装最新的长期支持版 Node.js。
3. 安装完成后，执行以下命令来使用这个新版本：
    
    ```
    nvm use [version_number]
    ```
    
    例如，如果它安装了 `20.14.0`，你就输入 `nvm use 20.14.0`。通常在 `nvm install` 之后，它会自动帮你切换。
4. **验证版本**：  
    执行以下命令来确认你的 Node.js 版本已经更新：
    
    ```
    node -v
    ```
    
    你应该会看到一个 `v20.x.x` 或更高的版本号。
