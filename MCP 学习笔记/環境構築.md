# 创建 MCP Server 所需的环境
==Windows操作系统==

1、如果你的电脑中没有安装`uv`工具，请用下面的命令进行安装。
```
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
请务必在之后重启终端，以确保`uv`命令生效。

2、为项目创建一个新目录，然后进入
```
mkdir first_mcp_project
cd first_mcp_project
```

3、使用 `uv` 初始化项目，创建虚拟环境并进入
```
uv init
uv venv
.venv\Scripts\activate
```

4、安装所需的依赖
```
uv add mcp[cli] httpx
uv add mcp arxiv
```

5、使用名为 inspector 的测试工具
```
npx @modelcontextprotocol/inspector uv run research_server.py
```

>[!note]
>执行上述命令可能会遇到失败的情况，请更新或切换 `node` 为最新的版本后再次运行。若出现提示信息询问“need to install the following packages”，请输入：`y`。

# 创建 MCP Client 所需的环境
==Windows操作系统==

1、如果你的电脑中没有安装`uv`工具，请用下面的命令进行安装。
```
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
请务必在之后重启终端，以确保`uv`命令生效。

2、为项目创建一个新目录，然后进入
```
mkdir first_mcp_project
cd first_mcp_project
```

3、使用 `uv` 初始化项目，创建虚拟环境并进入
```
uv init
uv venv
.venv\Scripts\activate
```

4、安装所需的依赖
```
uv add mcp anthropic python-dotenv nest_asyncio openai
```

5、配置环境变量
在中国国内只能使用国产的大模型，所以这里选择使用DeepSeek，通过 Anthropic API 调用 DeepSeek 模型，需要配置环境变量
```
export ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic
export ANTHROPIC_API_KEY=${YOUR_API_KEY}
```
